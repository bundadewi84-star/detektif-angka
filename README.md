# detektif-angka
Media pembelajaran interaktif untuk membongkar miskonsepsi nilai tempat bilangan pada siswa sekolah dasar.

## Deskripsi
Detektif Angka merupakan media pembelajaran interaktif yang dirancang untuk membantu siswa sekolah dasar memahami konsep nilai tempat bilangan serta mengidentifikasi miskonsepsi yang sering terjadi pada materi satuan, puluhan, dan ratusan.

## Fitur
- Materi interaktif
- Investigasi miskonsepsi
- Sistem skor dan predikat
- Sertifikat otomatis
- Download PDF sertifikat
- Responsif untuk laptop dan smartphone
